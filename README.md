===================================================
SHAHEEN: Autonomous Runway Sentinel
مشروع  - أكاديمية طويق
===================================================

1. نظرة عامة (Project Overview)
نظام "شاهين" هو حل تقني متقدم يعتمد على الطائرات بدون طيار (Drones) للمراقبة الآلية لمدرجات مطار الملك خالد الدولي. يهدف النظام إلى تعزيز السلامة الجوية من خلال الكشف التلقائي عن الأجسام الغريبة (FOD) وإدارة التهديدات البيولوجية (الطيور) باستخدام منطق طيران ذكي يعتمد على MAVSDK و PX4.

2. متطلبات التشغيل (System Prerequisites)
- نظام التشغيل: Linux (Native Environment)
- اللغة: Python 3.x
- المكتبات المطلوبة: pip install mavsdk asyncio
- المحاكي: PX4 SITL (Software In The Loop)

3. خطوات التشغيل (Launch Steps)

أولاً: تشغيل محاكي الطيران (Simulation Environment)
في نافذة تيرمينال جديدة، انتقل إلى مجلد الـ PX4-Autopilot وقم بتشغيل المحاكاة مع ضبط إحداثيات مدرج مطار الملك خالد:
export PX4_HOME_LAT=24.9777592
export PX4_HOME_LON=46.7020021
make px4_sitl jmavsim

ثانياً: تشغيل محطة التحكم (QGroundControl)
افتح برنامج QGroundControl لمراقبة الدرون والمسار بصرياً.

ثالثاً: تنفيذ كود المهمة (Mission Execution)
بعد استقرار المحاكي، قم بتشغيل الكود في تيرمينال أخرى:
python3 run_mission.py

4. الميزات التقنية (Technical Highlights)
- الطيران المتزامن: استخدام asyncio لإدارة مهام متعددة في وقت واحد.
- معايير السلامة: نظام Return to Launch (RTL) التلقائي عند انخفاض البطارية.
- الاستجابة للتهديدات: منطق ذكي للتعامل مع أسراب الطيور عبر الانحراف الآمن وتفعيل أنظمة الطرد.

5. هيكلية الكود (Code Structure)
- setup_mission_points(): توليد المسار الجوي للمدرج.
- check_drone_state(): دوال مراقبة حالة الطائرة.
- handle_fod() & handle_bird_flock(): دوال اتخاذ القرار الذكي عند رصد المعوقات.
